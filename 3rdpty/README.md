# Third Party Assets

These are files that I have collected for reference and/or the basis of my own derived works.

## [retrotechnology.com](https://www.retrotechnology.com/)
- Firmware from https://www.retrotechnology.com/herbs_stuff/ccs_nick.html
	- [ccsfdco.asm](https://www.retrotechnology.com/herbs_stuff/ccsfdco.asm) A version of the MOS 2.2 monitor from the manual for the 2422 FDC card.
	- [ccsfdc.asm](https://www.retrotechnology.com/herbs_stuff/ccsfdc.asm) A version of ccsfdco.asm modified by Nick Papadonis

- Links to many useful looking sites can be seen here:
	- [S-100 Links](https://retrotechnology.com/herbs_stuff/s100.html) Docs o'plenty!
	- [CalComp Links](https://www.retrotechnology.com/herbs_stuff/d_ccs.html) including a zip of CCS sources.

## [deramp.com](https://deramp.com/downloads/mfe_archive/010-S100%20Computers%20and%20Boards/00-California%20Computer%20Systems/10-CCS%20S100%20Boards/CCS%202422%20Floppy%20Controller/)
A California Computer Systems manual archive.

## [Z80-Retro!](https://github.com/Z80-Retro)
Acollection of Z80 and CP/M projects that provide much background on how one might develop a BIOS of one's own to run on a machine of this vintage.
